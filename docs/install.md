# Installetion guide:
## Linux:
### Get deb file:
> https://runserve.io/download/runserve.deb

### Get rpm file:
> https://runserve.io/download/runserve.rpm

## MacOS:
### Install using brew:

> brew tap runserve/runserve
> brew install runserve

## Updating to last stable version:
> runserve update
> runserve upgrade
